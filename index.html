Formularios

Los formularios son elementos que permiten recopilar información del usuario del sitio web.
Permite que el usuario ingrese un dato que después será utilizado por un Lenguaje de Programación.
Codigo HTML -> Permite indicar qué datos recopilar.
Código del Lenguaje de Programación -> Permite indicar qué hacer con esos datos recopilados.
PHP es un lenguaje de programación que se comunica con los servidores. Para usar PHP es necesario tener un servidor.

Elementos de Selección

En los elementos del selección el usuario responde a la pregunta: ¿Querés mandar este dato?
Si selecciona, responde que si.
Si no selecciona, responde que no.
El atributo se usa para indicar qué dato se manda cuando el usuario responde que si.

input:checkbox

Elementos de selección multiple. Permiten seleccionar uno, todos o ninguno.

input:radio

Elementos de selección simple. Permite solo la seleccion de uno de los elementos.
